# House-Price-Prediction
Developed a House Price Prediction model using Python, Pandas, Scikit-learn, and Gradio. Performed data preprocessing, feature engineering, model training, and evaluation to predict prices based on area, bedrooms, bathrooms, neighborhood, and year built. Deployed an interactive web app for real-time house price prediction.

# 🏠 House Price Prediction using Machine Learning

## 📖 Project Description
House Price Prediction is a Machine Learning project that predicts the price of a house based on its features. The model is trained using historical housing data and allows users to estimate house prices by entering property details through an interactive Gradio web application.

---

## 🎯 Objective
The objective of this project is to build an accurate machine learning model that predicts house prices using important house features such as area, number of bedrooms, bathrooms, neighborhood, and year built.

---

## 📊 Dataset Features

The dataset contains the following features:

| Feature | Description |
|---------|-------------|
| Square Footage | Total area of the house |
| Bedrooms | Number of bedrooms |
| Bathrooms | Number of bathrooms |
| Neighborhood | Location/Neighborhood of the house |
| Year Built | Construction year of the house |
| Price | Target variable (House Price) |
